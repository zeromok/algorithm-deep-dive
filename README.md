# Algorithm Deep Dive

알고리즘 문제를 풀며 발견한 새로운 기법이나 연습이 필요한 패턴을 정리하는 저장소입니다.

## 목적
- 문제 풀이에서 발견한 알고리즘 패턴을 독립적으로 연습
- 기법의 핵심 원리와 구현을 명확히 이해
- 재사용 가능한 패턴 코드로 정리

## 프로젝트 구조
```
src/main/java/daniel/algorithmdeepdive/
├── dataStructure/ # 자료구조 구현
├── mathematics/ # 수학 개념
└──  pattern/ # 알고리즘 패턴/기법
      ├── bruteforce/ # 완전탐색
      ├── dp/ # DP
      ├── greedy/ # 그리디
      ├── sort/ # 정렬
      ├── search/ # 탐색
```
